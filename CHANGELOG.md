# Changelog

All notable changes to The Sounds Player will be documented in this file.

## [2.0.0] - 2025-01-27

### Added
- **Playlist sidebar**: View and switch between all playlists without dropdown menus
- **Multi-playlist playback**: Playlists continue playing when switching to another
- **Bulk folder import**: Import entire folders of audio files at once
- **Playlist context menu**: Right-click to edit, import folder, or delete playlists
- **Visual playing indicator**: See which playlist is currently playing in the sidebar

### Changed
- Complete rewrite using Foundry VTT ApplicationV2 API
- Modern UI with sidebar layout
- Improved auto-save system

### Fixed
- All Foundry v13 deprecation warnings resolved

## [1.0.0] - 2025-01-24

### Added
- Initial release
- Multi-slot sound player
- Timeline seek functionality
- Custom slot images
- Drag & drop reordering
- Loop and volume controls
- Auto-save to Foundry playlists
